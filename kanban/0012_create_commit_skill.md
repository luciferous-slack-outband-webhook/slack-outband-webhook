# `git commit`するスキルかコマンドを追加して欲しい
## 要望

gitにおいて、ステージング状態のファイルを読み取り、コメント(titleとdescription)を生成しコミットして欲しい。
スキルかコマンドとしてほしい (`/commit`みたいな感じで)。

## プラン

### 概要
`.claude/commands/commit.md` を新規作成し、`/commit` スラッシュコマンドを追加する。

### コマンド動作
1. **状態確認**: `git status`, `git diff --cached`, `git log` を並列実行
2. **メッセージ生成**: ステージされた変更内容からコミットメッセージ（title + description）を自動生成
3. **コミット実行**: HEREDOC 形式で `git commit` を実行（Co-Authored-By 付き）
4. **結果確認**: `git status` で成功確認

### 設計方針
- 既にステージされたファイルのみを対象とする（新たなステージングは行わない）
- 引数でコミットメッセージを指定可能（指定がない場合は自動生成）
- 秘密情報を含む可能性があるファイルがステージされている場合は警告
- pre-commit hook 失敗時は問題を修正して新規コミット（amend しない）

### 対象ファイル
- **新規作成**: `.claude/commands/commit.md`
