# `/kanban` スキルのセッション名を `{xxxx}_{title}` 形式にする

## 目的
セッション名と対象のカンバンが一致するようにして認知負荷を下げたい

## 要望
`/kanban` スキルにおいてセッション名を `{xxxx}_{title}` にして欲しい。

---

## 完了サマリー

- **完了日時**: 2026-04-21T22:04:24+09:00
- **対応内容**:
  - 調査の結果、Claude Code にはスキルからセッション名を自動変更する手段がないことが判明
  - ユーザー判断により「実現不可」としてタスクをクローズ
- **調査根拠**:
  - `claude --name` フラグはセッション起動時のみ有効（既存セッションには不可）
  - `/rename` コマンドはユーザーがプロンプトに入力する必要があり、スキル/アシスタントから呼び出す手段はない
  - Skill ツールから呼べる組み込みコマンドのホワイトリストは `/init`, `/review`, `/security-review` のみで `/rename` は含まれない
  - 出典: https://code.claude.com/docs/en/cli-reference 、https://code.claude.com/docs/en/commands
- **変更ファイル**:
  - なし（コード・スキル・設定ファイルの変更なし）
